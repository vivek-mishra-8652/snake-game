# 🐍 Snake Game

A classic Snake game built with Python and Pygame-ce.

## Features
- Smooth animated gameplay
- Score tracking
- Game over screen
- Keyboard controls

## Installation
pip install pygame-ce

## How to Run
python snake_game.py

## Controls
| Key | Action |
|-----|--------|
| W / Arrow Up | Move Up |
| S / Arrow Down | Move Down |
| A / Arrow Left | Move Left |
| D / Arrow Right | Move Right |
| R | Restart |
| Q | Quit |

## Tech Stack
- Python 3.14
- Pygame-ce
